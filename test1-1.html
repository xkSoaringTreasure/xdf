    # 根据学员号查询学员报班记录查询

    • url: http://wx.jn.xdf.cn/api/tools/get_classes_by_stuno 点击进行在线测试

    # 发送数据：

    字段 字段描述 是否必须 字段参考
    student_code 学员号 是 JN299759
    school_id 学校id 默认 22

    # 返回信息：list

    字段 字段描述 字段参考
    classCode 班号 LWY1903
    className 班级名称 六级无忧计划
    classStartDate 开课时间 2019-04-13 00:00:00
    classEndDate 结课时间 2019-05-26 00:00:00
    classAddress 校区 凯旋校区
    lessonCount 课时 28
    classSprintTime 上课时间 每周六日下午14:00-16:00;16:20-18:20
    schoolId 学校id 22
    valid 1
