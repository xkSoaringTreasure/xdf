    # 根据学员openid获取学员信息

    • url: http://wx.jn.xdf.cn/api/tools/message 点击进行在线测试

    # 发送数据：

    字段 字段描述 是否必须 字段参考
    open_id openid 是 oboK3wPALOV9lsPhk8zUkWD6v8FI
    school_id 学校id 默认 22

    # 返回信息：object

    字段 字段描述 字段参考
    bindingid 绑定id CB18901D-B840-4785-9FEA-EB58892D5E84
    school_id 学校id 22
    studentname 学员姓名 方聪瑞
    studentmoblie 学员手机号 18660807489
    studentcode 学员id JN265043
    bindingdate 绑定时间戳 1553050246000
    openid openid oboK3wPALOV9lsPhk8zUkWD6v8FI
    isdefault 是否默认 1
